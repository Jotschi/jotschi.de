    Canon MX300 with Debian Lenny

    Canon MX300 unter Debian Lenny

    Mittels CVS Version von Sane (22.05.2008)
    1. Sane auschecken, bauen und installieren.

    Using the CVS version of Sane (22.05.2008)
    1. checkout sane, build and install.

      cvs -z3 -d:pserver:anonymous@cvs.alioth.debian.org:/cvsroot/sane co sane-backends

    2. Gruppe ‘scanner’ anlegen (falls dies nicht schon gemacht wurde)
    3. Der Benutzer der scannen darf muss der Gruppe ‘scanner’ hinzugefügt werden.
    4. UDev Rule anlegen
    5. Scanner anschließen
    6. xsane ausführen

    2. Add group ‘scanner’ (if not yet been done already)
    3. The user that scans must be added to the group ‘scanner’.
    4. Add matching UDev rule.
    5. Connect your scanner.
    6. execute xsane


    uDev Rule:

    

      
	
    1. Canon PIXMA MX300
      
SYSFS{idVendor}"04a9", SYSFS{idProduct}“1727”, MODE=“0664”, GROUP=“scanner”, ENV{libsane_matched}=“yes”
